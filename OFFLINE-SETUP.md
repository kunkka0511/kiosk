# Mandala Garden Kiosk — Offline ажиллуулах заавар

Kiosk-ийн **үндсэн дэлгэц бүрэн локал** (интернет шаардахгүй). "Offline ажиллана" гэдэг нь
аппликейшнийг интернетээс биш, **kiosk компьютер дээрээ** ажиллуулна гэсэн үг.

> **Гол дүрэм:** Нэг удаа интернеттэйгээр `build` хий → дараа нь үүрд offline ажиллана.

---

## 1. Нэг удаагийн бэлтгэл (ИНТЕРНЕТТЭЙ үед)

Kiosk компьютер дээр (эсвэл build хийгээд folder-оо хуулна):

```bash
npm install          # node_modules + Google фонт татна (зөвхөн энэ үед интернет хэрэгтэй)
npm run build        # бүх зүйлийг offline-д бэлэн болгоно
```

`npm run build` хийсний дараа дараах зүйлс **өөр дээр нь хадгалагдана** (интернет цаашид хэрэггүй):
- Google фонт (Inter, Playfair) — Next.js өөртөө татаж self-host хийнэ
- Брэнд фонт, зураг, видео — бүгд `public/`-д
- API (`/api/toim`, `/api/amenities`) — локал, файлын системээс уншина

---

## 2. Offline ажиллуулах (ИНТЕРНЕТГҮЙ)

```bash
npm run start        # http://localhost:3000 дээр локал сервер
```

Дараа нь хөтчөө `http://localhost:3000` руу нээнэ. Болоо — интернет огт хэрэггүй.

**Хялбар арга:** `start-kiosk.bat`-г 2 дарж асаа — сервер + Chrome-г бүтэн дэлгэцэд автоматаар нээнэ.

---

## 3. Автоматаар асаах (компьютер унтарч асах бүрт)

1. `Win + R` → `shell:startup` → нээгдсэн folder-т `start-kiosk.bat`-ийн **shortcut** хий.
2. Компьютер асах бүрт сервер + kiosk автоматаар нээгдэнэ.

**Найдвартай байдал:** Аппликейшн дотроо `ErrorBoundary`-тэй — алдаа гарвал 8 секундын дараа
өөрөө reload хийнэ. Сервер унтарвал дахин асаахын тулд Windows **Task Scheduler**-т
"restart on failure" тохируулж болно.

---

## 4. Анхаарах зүйлс

- **Static export (`output: export`) БҮҮ ашигла** — `/api/toim`, `/api/amenities` нь folder-оос
  зураг уншдаг тул динамик сервер хэрэгтэй. Одоогийн `output: standalone` тохиргоо зөв.
- **Шинэ зураг нэмэх:** `public/assets/toim/` эсвэл `public/assets/amenities/<dir>/`-т зураг хийнэ.
  Сервер ажиллаж байвал шинэчлэхэд шууд гарч ирнэ (build дахин хийх шаардлагагүй).
- **QR код:** `CONTACT.url` нь зочдын утсаар уншигдана (тэдний интернет) — kiosk-д нөлөөлөхгүй.
- **`/effects` хуудас** picsum.photos (гадны зураг) ашигладаг ч kiosk үүнийг дуудахгүй.

---

## 5. (Сонголт) Жижиг standalone хувилбар

`output: standalone` тул build хийхэд `.next/standalone/` дотор бүгдийг багтаасан жижиг
сервер үүснэ. Үүнийг бүтэн `node_modules`-гүйгээр хуулж ажиллуулж болно:

```bash
npm run build
# .next/standalone/ дотор:  .next/static болон public-аа хуулж тавиад:
node server.js
```
